# 学習曲線分析 — 進捗タスクリスト

5-fold CV × 6段階の訓練サイズで学習曲線を描く。  
合計 **30ジョブ**。Colabで1ジョブずつ実行して結果をDriveに蓄積する。

## 実行手順

1. `00_prepare_folds.ipynb` — フォールド割り当てを生成（**1回だけ**）
2. `01_train_one_job.ipynb` — `FOLD` と `SIZE` を指定して1ジョブ実行
3. `02_plot_curve.ipynb` — 全結果を集計して学習曲線をプロット
